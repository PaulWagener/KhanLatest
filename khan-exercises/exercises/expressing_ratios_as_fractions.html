person(1) has plural( NUM, fruit( 0 ) ) for every plural( DENOM, fruit( 1 ) ). Write the ratio of fruit( 0 )s to fruit( 1 )s as a simplified fraction.

NUM / DENOM

Verhoudingen kunnen op verschillende manieren worden geschreven, maar betekenen hetzelfde.

You can express a ratio with a colon separating the two numbers.

NUM:DENOM

Zo kun je het opschrijven als een zin.

NUM \text{ to } DENOM

Of je kunt een verhouding opschrijven als een breuk.

\dfrac{NUM}{DENOM}=fractionReduce( NUM, DENOM )

Daarom, fractionReduce( NUM, DENOM ) is de verhouding van fruit( 0 )s tot fruit( 1 )s geschreven als een versimpelde breuk.
